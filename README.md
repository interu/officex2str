# Officex2str

Convert Microsoft office 2007/2010 files(docx/xlsx/pptx) to string.

## Installation

Add this line to your application's Gemfile:

    gem 'officex2str'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install officex2str

## Usage

    Officex2str.convert("file_path")

## Rspec

    $ bundle exec rake
